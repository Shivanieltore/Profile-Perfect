# 📄Resume Builder: Create Professional Resumes & CVs

Resume Builder is a web-based application that allows users to create, customize, and download professional resumes and CVs using pre-designed templates.

---

## 🚀 Features
-Multiple resume & CV templates
-Personal Info Form to auto-fill details across templates
-CV Maker Page with customization options
-Export resumes in Word (.docx) and PDF formats
-Simple and interactive web UI

---

## 🛠️ Tech Stack
-Frontend: HTML, CSS, JavaScript
-Backend: Flask / Node.js (choose depending on your setup)
-Database: MongoDB / SQLite (if storing user data)
-Libraries/Tools:
-python-docx / reportlab → For Word & PDF generation
-Flask / Express.js → For server-side handling
-Bootstrap / Tailwind → For responsive design
